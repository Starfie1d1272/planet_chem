## 任务概述
本次作业为《行星化学》课程的第一次平时作业，主要涉及核天体物理的基础理论与同位素放射性衰变计算。作业解答形式以手写理论推导为主，并在超新星光变曲线的计算中结合了 Python 编程建模与可视化分析。

### 涉及的核心知识点：
1. 恒星内部的核合成反应路径分析。
2. 核素图上的 s-过程（慢中子捕获）与 r-过程（快中子捕获）路径演化。
3. 放射性同位素的半衰期计算与活度推导。
4. 基于纯放射性衰变（56Ni -> 56Co -> 56Fe）的超新星（SN 1987A）理论光变曲线数值模拟。
