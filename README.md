## 仓库简介
本仓库用于记录与归档南京大学《行星化学》课程的平时作业、理论推导报告以及相关的 Python 数据分析与可视化代码。

《行星化学》课程的作业主要侧重于核天体物理、同位素年代学、化学热力学及量子化学等领域的物理化学计算。因此，本仓库的结构设计兼顾了理论推导的严谨性与数值计算的可复现性，包含了手写推导扫描件、LaTeX 电子版报告以及对应的 Python 脚本程序。

## 方法论与技术栈
本仓库的所有编程任务均在标准化的 Python 环境下开发测试。主要依赖如下：

1. 理论推导：作业中的化学热力学、量子化学及核衰变公式均通过严谨的 LaTeX 语法进行排版记录，早期作业保留手写扫描件以供对照。
2. 数值计算：主要依托 Python 科学计算生态（Numpy, SciPy, Matplotlib）进行配分函数的级数累加、同位素衰变建模以及反应平衡常数的求解。
3. 数据来源：计算中涉及的标准摩尔焓变、熵变等热力学数据主要参考并引用自 NIST Chemistry WebBook。

## 目录规范
为保持仓库整洁，每次作业均采用独立的子文件夹进行隔离封装。典型的单次作业目录结构如下：
* `HWX_Questions.pdf`: 原始作业题目归档。
* `HWX_Report.md` / `.pdf`: 正式提交的学术报告或手写扫描件。
* `*.py` / `*.ipynb`: 该次作业涉及的数值分析代码核心逻辑。
* `figures/`: 运行代码生成的科学可视化图件合集。
